# Thaibhavan PG Website

A premium, minimal, and highly engaging landing page for **Thaibhavan PG**, a premier ladies' homestay located in Alappuzha, Kerala—specifically tailored for CA college students.

## ✨ Features

- **🎯 Niche Focused**: Designed specifically for the needs of CA students, highlighting peaceful study environments and safety.
- **🖼️ Adaptive Gallery**: High-end horizontal carousel that respects image aspect ratios (no cropping) with smooth swipe/scroll support.
- **⚡ Supercharged Performance**: Built with **React 18** and **Vite** for near-instant load times.
- **🎨 Premium Aesthetic**: Styled with **Tailwind CSS v4** featuring a clean, minimal "Filmstrip" design with glassmorphism effects.
- **🎬 Fluid Animations**: Powered by **Framer Motion** for spring-based reveals and engaging micro-interactions.
- **📱 Mobile-First**: Fully responsive across all devices—from small smartphones to large desktops.
- **📞 Action-Oriented**: Integrated WhatsApp API, direct call links, and email triggers for seamless bookings.

## 🛠️ Technology Stack

- **React 18**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion**
- **Lucide Icons**

## 🚀 Getting Started

### 1. Installation
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

The production-ready files will be available in the `dist/` folder.

## 📁 Project Structure

- `src/components/`: Modular UI components (Gallery, Hero, Contact, etc.)
- `public/`: High-resolution property images and assets.
- `src/index.css`: Global design tokens and Tailwind configuration.

## 🌍 Deployment

Optimized for instant deployment on **Netlify** or **Vercel**. Simply upload the `dist/` folder to your preferred hosting provider.
